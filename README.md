# BePro

**The Career Engine for Developers Who Build in Silence.**

BePro is a platform built for developers who care more about output than optics.

It’s not a learning app.  
It’s not another resume network.  
It’s not built for people who want validation.  

BePro is for the top 5% — the ones who show up, ship work, and level up with focus.  
AI-generated roadmaps. Weekly missions. Verified college communities. Proof-of-work profiles.  
No noise. No vanity. Just forward motion.

---

## What BePro Does

**Codex (AI Roadmaps)**  
Get personalized, role-specific learning plans that break down top-tier tech roles into weekly missions.

**Weekly Missions**  
Clear, focused tasks with curated resources — you build, submit, and track everything.

**Proof-of-Work Profiles**  
Your profile reflects what you've done, not just where you’ve studied or worked.

**Verified College Communities**  
Private career networks per university. You’re only inside if your college email is verified.

**Explore Feed**  
AI-curated feed that surfaces high-signal work. Interactions are tracked. Discovery is optimized.

## Getting Started

This project runs on **free-tier infrastructure**, so expect **cold starts** and some **latency**. That’s the cost of staying independent.

## Tech Stack

- **Frontend**: Next.js 15, React 19, Tailwind 4
- **Backend**: Supabase (Postgres, Auth, Realtime)
- **UI**: Radix UI, Lucide Icons
- **Realtime**: Socket.io
- **State**: Zustand
- **Animation**: Framer Motion
